## Execute all tests in this directory
The tests are split into groups; each could be executed by different CI jobs in parallel.

```
    cd /path/to/fabric/test/regression/weekly
```

```
    ./runGroup1.sh
    ./runGroup2.sh
    ./runGroup3.sh
```
